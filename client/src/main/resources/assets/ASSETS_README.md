# Placeholder Assets

This folder contains placeholder images for games without covers.

## Required Assets
- `placeholder_cover.png` - 600x900 placeholder cover image
- `placeholder_hero.png` - 1920x1080 placeholder hero/banner image

## Note
You can replace these with your own placeholder images or use external URLs.
The PlaceholderMetadataService will attempt to use Steam CDN URLs for Steam games automatically.
