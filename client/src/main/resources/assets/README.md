# Nexus Launcher Assets

This folder contains application assets:

- `icon.png` - Application icon (add your own 256x256 PNG)
- `placeholder.png` - Placeholder image for games without covers
- `logo.png` - Nexus Launcher logo

## Adding Assets

Place your assets in this folder and reference them in code using:
```java
getClass().getResourceAsStream("assets/icon.png")
```
